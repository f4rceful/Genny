# MathBox

**MathBox** — это современное веб-приложение для выполнения арифметических вычислений, обладающее стильным интерфейсом в тёмных тонах и функционалом ведения истории операций.

## Реализованные функции
- **ФТ-01 (Базовые операции):** Сложение, вычитание, умножение и деление через удобный интерфейс.
- **ФТ-02 (Сброс):** Мгновенная очистка текущего состояния кнопкой «C».
- **ФТ-03 (Обработка ошибок):** Безопасное деление на ноль с выводом предупреждения на русском языке и красной подсветкой.
- **ФТ-04 (Веб-доступность):** Чистый Vanilla JS/CSS/HTML без зависимостей.
- **ФТ-05 (История):** Автоматическое сохранение до 10 последних операций в `localStorage` с защитой от XSS.

## Особенности
- **Тёмная тема:** Эстетичный UI, снижающий нагрузку на глаза.
- **Копирование:** Кнопка быстрого копирования результата в буфер обмена.
- **Адаптивность:** Корректная работа на мобильных устройствах и десктопах.
- **Клавиатура:** Поддержка ввода цифр и операций с физической клавиатуры.

## Запуск
1. Склонируйте репозиторий или скачайте файлы.
2. Откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Safari, Edge).
3. Дополнительная установка зависимостей или запуск сервера не требуется.
