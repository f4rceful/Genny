## Нефункциональные требования

### НФТ-01: Время загрузки курсов валют
**Категория:** Производительность
**Описание:** Система должна загружать актуальные курсы валют из внешнего API не дольше 5 секунд. При превышении таймаута система должна немедленно переключаться на использование кешированных данных (без ожидания ошибки от API).
**Источник:** БТ-02, БТ-06

### НФТ-02: Совместимость с современными браузерами
**Категория:** Совместимость
**Описание:** Веб-интерфейс системы должен корректно работать в последних версиях браузеров Google Chrome, Mozilla Firefox, Safari и Яндекс.Браузер. Недопустимо использование технологий, требующих установки дополнительных плагинов или расширений.
**Источник:** БТ-08

### НФТ-03: Удобство использования (юзабилити)
**Категория:** Удобство использования
**Описание:** Интерфейс должен соответствовать следующим критериям: автоматическое обновление результата при любом изменении входных данных (без кнопки «Конвертировать»), интуитивно понятная смена валют одной кнопкой «⇄», явная индикация ошибок красным цветом, отображение статуса загрузки курсов, поддержка тёмной темы по умолчанию. Все тексты интерфейса должны быть на русском языке.
**Источник:** Features, БТ-04, БТ-03

### НФТ-04: Безопасность обработки пользовательского ввода
**Категория:** Безопасность
**Описание:** При отображении истории конвертаций, сохранённой в localStorage, система должна исключать выполнение вредоносного кода (XSS). Все данные, введённые пользователем (сумма, выбранные валюты), должны проходить санитизацию перед выводом на страницу. Недопустима непосредственная вставка пользовательского ввода в DOM без экранирования.
**Источник:** БТ-07, Общепринятая практика

### НФТ-05: Надёжность при недоступности внешнего API
**Категория:** Надёжность
**Описание:** При отсутствии подключения к сети, ошибках HTTP или таймауте запроса к внешнему API система должна использовать кешированные курсы валют, сохранённые в localStorage, и уведомлять пользователя о дате последнего успешного обновления. Если кеш отсутствует или устарел (TTL 1 час), система должна блокировать функциональность конвертации и отображать сообщение о временной недоступности сервиса.
**Источник:** БТ-06, UC-03
