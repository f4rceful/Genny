# Юз-кейсы

### UC-01: Успешная конвертация валюты

**Источник:** БТ-01, БТ-02, БТ-03, БТ-08
**Актор:** Пользователь
**Предусловие:** Веб-страница конвертера открыта в браузере. Актуальные курсы валют загружены или получены из кеша.

**Основной поток:**
1. Пользователь выбирает исходную валюту (например, USD) из выпадающего списка.
2. Пользователь выбирает целевую валюту (например, EUR) из выпадающего списка.
3. Пользователь вводит сумму для конвертации в соответствующее поле ввода.
4. Система автоматически отображает результат конвертации.
5. Система сохраняет запись о проведенной операции в историю конвертаций (localStorage).

**Альтернативные потоки:**
- Некорректный ввод: Если пользователь вводит пустое значение, ноль или отрицательное число, система отображает сообщение об ошибке (подсвеченное красным) и не показывает результат конвертации.
- Выбор одинаковых валют: Если пользователь выбирает одинаковую исходную и целевую валюту, система отображает результат, равный введённой сумме (курс 1:1), и сохраняет операцию в историю.

**Постусловие:** Результат конвертации отображен. Запись о конвертации добавлена в историю.

### UC-02: Смена направления конвертации

**Источник:** БТ-04
**Актор:** Пользователь
**Предусловие:** Веб-страница конвертера открыта. Пользователь выбрал исходную и целевую валюты.

**Основной поток:**
1. Пользователь нажимает кнопку «⇄» (поменять местами) между полями выбора валют.
2. Система меняет местами значения исходной и целевой валюты.
3. Система автоматически пересчитывает и отображает результат конвертации для новой пары валют на основе введенной суммы.
4. Система сохраняет запись о новой операции в историю конвертаций.

**Альтернативные потоки:**
- Некорректный ввод: Если в поле суммы введено некорректное значение (пусто, ноль, отрицательное число), система после смены валют продолжает отображать сообщение об ошибке и не показывает результат конвертации.

**Постусловие:** Исходная и целевая валюта поменяны местами. Отображен новый результат конвертации. Новая запись добавлена в историю.

### UC-03: Использование кешированных курсов валют

**Источник:** БТ-06
**Актор:** Пользователь
**Предусловие:** Веб-страница конвертера открывается или обновляется. Внешний API (https://open.er-api.com/v6/latest/USD) временно недоступен.

**Основной поток:**
1. Система пытается загрузить актуальные курсы валют из внешнего API.
2. Система обнаруживает ошибку соединения или таймаут запроса.
3. Система загружает последние успешно полученные курсы из кеша (localStorage) вместе с меткой времени их сохранения.
4. Система отображает интерфейс конвертера, используя кешированные курсы.
5. Система отображает информационное сообщение: «Курс от ДД.ММ.ГГГГ. Актуальные данные временно недоступны».
6. Пользователь может производить конвертацию, используя закешированные данные.

**Альтернативные потоки:**
- Кеш отсутствует или устарел: Если кешированных данных нет или их срок жизни (TTL) превысил 1 час, система отображает сообщение о временной недоступности сервиса и блокирует функциональность конвертации.

**Постусловие:** Конвертер работает на устаревших, но доступных данных. Пользователь уведомлен о статусе данных.
