# CurrencyFlow

CurrencyFlow — это современное веб-приложение для мгновенной конвертации валют. Приложение ориентировано на скорость работы, надежность и удобство использования в темной теме.

## Особенности модуля
- **Автоматический расчёт**: Результат обновляется мгновенно при вводе суммы или смене валюты.
- **Умное кеширование**: Актуальные курсы сохраняются в `localStorage`. Если API недоступен, приложение использует кеш (если он не старше 1 часа).
- **История операций**: Хранит 10 последних успешных конвертаций.
- **Безопасность**: Защита от XSS при отображении пользовательских данных из истории.
- **Тёмная тема**: Интерфейс спроектирован для комфортной работы в условиях низкой освещенности.

## Запуск
Поскольку приложение использует `fetch()` для получения данных из внешнего API, для корректной работы в некоторых браузерах может потребоваться запуск через локальный сервер (из-за политик CORS или ограничений `file://`).

1. Склонируйте или скачайте файлы проекта.
2. Запустите локальный сервер (например, используя расширение VS Code "Live Server" или команду `python -m http.server`).
3. Откройте `index.html` в браузере.

## Тесты
Для ручного тестирования функционала:
1. **ФТ-01/05**: Проверьте наличие USD, EUR, RUB, KZT в списках.
2. **ФТ-03**: Введите "100" в поле суммы — результат должен появиться мгновенно.
3. **ФТ-04**: Нажмите кнопку `⇄` — валюты поменяются местами, результат пересчитается.
4. **ФТ-06**: Отключите интернет и перезагрузите страницу — приложение должно показать сообщение "Курс от [Дата]" и продолжить работу.
5. **ФТ-07**: Введите отрицательное число или буквы — появится красная ошибка.

## API
Приложение использует бесплатный публичный API: [Exchangerate-API](https://www.exchangerate-api.com/).
- **Endpoint**: `https://open.er-api.com/v6/latest/USD`
- **Ключ**: Не требуется для стандартных запросов.
- **Отказоустойчивость**: Реализован таймаут 5 секунд. Если за это время API не отвечает, система автоматически переходит на использование данных из локального кеша.
- **Индикация**: Статус загрузки (актуально/из кеша/ошибка) отображается в верхней части приложения.
