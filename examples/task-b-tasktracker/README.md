# TaskFlow

TaskFlow — это современное SPA-приложение для управления задачами на базе методологии Канбан. Оно позволяет организовывать работу над проектами, отслеживать дедлайны и анализировать продуктивность через удобный дашборд.

## Запуск

1. Склонируйте или скачайте проект.
2. Откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Safari, Edge).
3. Приложение не требует сервера и подгружает данные из локального хранилища браузера.

## Реализованные функции

- **Управление проектами**: Создание проектов с цветовой идентификацией.
- **Канбан-доска**: 4 колонки (Бэклог, В работе, На проверке, Готово) с автоматическим подсчетом задач.
- **Управление задачами**: Создание, редактирование, удаление и перемещение задач между этапами одной кнопкой.
- **Аналитика (Дашборд)**: Визуализация общего прогресса, статистики по статусам и списка ближайших дедлайнов.
- **Умная фильтрация**: Живой поиск, фильтрация по приоритетам и динамически собираемым тегам.
- **Контроль сроков**: Визуальное выделение просроченных задач и задач на сегодня.
- **Локальное хранение**: Автоматическое сохранение всех данных в `localStorage`.
- **Представления**: Переключение между видом канбан-доски и табличным списком.

## Технологии

- HTML5
- CSS3 (переменные, Flexbox, Grid)
- Native JavaScript (ES6+ Классы)

## Совместимость

Приложение оптимизировано для настольных браузеров с разрешением экрана от 1024px. Тестировалось в последних версиях браузеров на движке Chromium.
