# Бизнес-процесс (БП) — Конвертация валюты

**Актор:** Пользователь

## Основной поток

1. Пользователь открывает веб-страницу конвертера.
2. Пользователь выбирает исходную валюту из выпадающего списка (например, USD).
3. Пользователь выбирает целевую валюту из выпадающего списка (например, EUR).
4. Пользователь вводит сумму в поле ввода.
5. Система автоматически отображает результат конвертации.
6. Система сохраняет запись в историю конвертаций.

## Альтернативный поток — смена направления

4а. Пользователь нажимает кнопку «⇄» (поменять местами).
4б. Система меняет местами исходную и целевую валюты и пересчитывает результат.

## Альтернативный поток — некорректный ввод

4а. Пользователь вводит пустое значение, ноль или отрицательное число.
4б. Система отображает сообщение об ошибке и не показывает результат.

## Альтернативный поток — одинаковые валюты

3а. Пользователь выбирает одинаковую исходную и целевую валюту.
3б. Система отображает результат равный введённой сумме (курс 1:1).
