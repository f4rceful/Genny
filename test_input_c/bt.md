# Бизнес-требования (БТ) — Конвертер валют

| ID    | Требование                                                                                                              | Обязательность |
|-------|-------------------------------------------------------------------------------------------------------------------------|----------------|
| БТ-01 | Пользователь должен иметь возможность выбрать исходную валюту и целевую валюту из списка                                | Обязательное   |
| БТ-02 | Пользователь должен иметь возможность ввести сумму для конвертации                                                      | Обязательное   |
| БТ-03 | Система должна отображать результат конвертации с точностью до 2 знаков после запятой                                   | Обязательное   |
| БТ-04 | Пользователь должен иметь возможность поменять исходную и целевую валюты местами одним нажатием                         | Обязательное   |
| БТ-05 | Система должна корректно обрабатывать некорректный ввод (пустое поле, отрицательные числа, нули)                        | Обязательное   |
| БТ-06 | Пользователь должен видеть историю последних 10 конвертаций в текущей сессии                                            | Опциональное   |
| БТ-07 | Интерфейс должен быть доступен через веб-браузер без установки дополнительного ПО                                       | Обязательное   |
