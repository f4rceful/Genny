# Характеристики (Features)

- Название приложения — «CurrencyFlow»
- Тёмная тема по умолчанию (тёмный фон, светлый текст)
- Поддерживаемые валюты: USD, EUR, RUB, GBP, CNY, JPY, CHF, AED — курсы хранятся статично в JS
- Базовая валюта для курсов — USD
- Кнопка «⇄» между полями валют для быстрой смены направления конвертации
- Результат обновляется автоматически при изменении суммы или выборе валюты (без кнопки «Конвертировать»)
- История конвертаций отображается под формой (последние 10 операций): «100 USD → 91.50 EUR»
- При некорректном вводе сообщение об ошибке подсвечивается красным
- Данные истории хранить в localStorage между сессиями
- Все надписи на русском языке
