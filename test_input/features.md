# Характеристики (Features)

- Название приложения — «MathBox»
- Тёмная тема по умолчанию (тёмный фон, светлый текст)
- Кнопка копирования результата в буфер обмена (рядом с полем результата)
- История вычислений отображается в боковой панели (последние 10 операций)
- При вводе невалидного значения (например, деление на ноль) сообщение об ошибке подсвечивается красным
- Все надписи на русском языке
- Данные истории хранить в localStorage между сессиями
