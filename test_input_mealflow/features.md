# Характеристики (Features) — MealFlow

- Название приложения — «MealFlow»
- Светлая тема по умолчанию (фон #f8fafc, карточки #ffffff, акцент #10b981 — изумрудный, вторичный #f59e0b — янтарный)
- SPA на чистом HTML + CSS + JS (без фреймворков и сборщиков)
- Все тексты на русском языке

---

## Навигация

Боковая панель (sidebar, 220px) с иконками и подписями:
- 🍽 Рецепты
- 📅 План питания
- 🛒 Список покупок
- 📊 Анализ питания
- 🥦 База ингредиентов

Активный раздел выделен цветом акцента. На иконке «Список покупок» — бейдж с количеством некупленных позиций (если > 0).

---

## Раздел «Рецепты»

**Галерея** карточек в сетке 3 колонки (адаптивно: 2 при узком окне):
- Фото (или placeholder с эмодзи кухни)
- Название, кухня, время, сложность (цветные бейджи)
- Калорийность на порцию
- Иконка ❤ (избранное) в правом верхнем углу — активная/неактивная
- При наведении — кнопки «Открыть», «✎», «🗑»

**Панель фильтров** над галереей (в 1 строку):
- Input поиска
- Select кухни (Все / Русская / Итальянская / Азиатская / Средиземноморская / Мексиканская)
- Select тега
- `<input type="range">` для времени (отображается текущее значение рядом: «до 45 мин»)
- `<input type="range">` для калорий/порцию (отображается текущее значение: «до 600 ккал»)
- Toggle ❤ «Избранные»
- Кнопка «Сбросить»

**Детальный вид рецепта** (открывается в модале или в правой панели):
- Фото (крупное), название, мета-информация
- Спиннер порций с кнопками «−» / «+» и вводом числа — пересчёт мгновенный
- Таблица ингредиентов: количество (пересчитанное) | единица | название | КБЖУ строки (если ингредиент есть в базе)
- Итоговые КБЖУ на выбранное кол-во порций
- Кнопка «Добавить в план» — открывает выбор дня и приёма пищи
- Шаги приготовления (нумерованный список)

---

## Раздел «План питания»

**Недельная сетка**: 7 колонок (Пн–Вс), 4 строки (Завтрак, Обед, Ужин, Перекус).
- Заголовок каждой колонки: день недели + суммарные ккал/день (цветом: зелёный если <2000, жёлтый 2000–2500, красный >2500)
- Каждая ячейка — вертикальный список блоков рецептов + кнопка «+»
- Блок рецепта в ячейке: название (укороченное), «×N порций», ккал, кнопка «✕»
- Горизонтальный скролл при необходимости

---

## Раздел «Список покупок»

- Заголовок с кнопками «Сбросить все» и «Очистить купленные»
- Аккордеон по категориям (каждая раскрывается/сворачивается, счётчик непокупленных в заголовке)
- Строка позиции: чекбокс | название ингредиента | количество + единица | категория
- Купленные: зачёркнутый текст, серый фон, перемещаются в конец группы
- Если список пуст — иллюстрированная заглушка

---

## Раздел «Анализ питания»

**Метрики недели** (4 карточки горизонтально): ккал, белки г, жиры г, углеводы г — с иконкой и цветовым акцентом.

**SVG-диаграмма 1 — столбчатая «Калории по дням»**:
- 7 столбцов (пн–вс), высота пропорциональна калориям
- Горизонтальная пунктирная линия нормы 2000 ккал с подписью
- Подписи значений над столбцами
- Цвет столбца: зелёный (<2000), жёлтый (2000–2500), красный (>2500)
- Всё на чистом SVG, без библиотек

**SVG-диаграмма 2 — круговая «Распределение БЖУ»**:
- 3 сектора: белки (синий), жиры (жёлтый), углеводы (зелёный)
- Легенда с граммами и процентами
- Анимация появления через CSS stroke-dasharray

**Таблица по дням**: 7 строк, для каждого дня — список приёмов пищи с КБЖУ.

---

## Раздел «База ингредиентов»

- Таблица: название | категория | ккал/100г | Б | Ж | У | действия
- Поиск по названию
- Кнопка «+ Ингредиент» → модальная форма
- Редактирование/удаление (с предупреждением если ингредиент используется в рецептах)

---

## Алгоритм агрегации списка покупок

Конвертация при суммировании:
- г + г = г; г + кг → всё в г, итог если ≥1000 г → перевести в кг
- мл + мл = мл; мл + л → всё в мл, итог если ≥1000 мл → перевести в л
- шт + шт = шт
- ст.л. + ст.л. = ст.л.
- ч.л. + ч.л. = ч.л.
- щепотка — просто считается количество щепоток
- Разные несовместимые единицы одного ингредиента → отдельные строки

## Демо-данные (при первом запуске)

**6 рецептов:**
1. Овсяная каша с бананом (Завтрак, Русская кухня, 10 мин, Лёгкий)
2. Паста Карбонара (Обед, Итальянская, 25 мин, Средний)
3. Греческий салат (Обед/Ужин, Средиземноморская, 15 мин, Лёгкий)
4. Куриный суп с лапшой (Обед, Русская, 60 мин, Средний)
5. Том Ям с морепродуктами (Обед/Ужин, Азиатская, 40 мин, Сложный)
6. Шоколадный смузи (Перекус, 5 мин, Лёгкий)

**База ингредиентов (≥20 позиций)** с реальными КБЖУ: овсянка, банан, яйца, бекон, пармезан, паста, огурец, томаты, маслины, сыр фета, оливковое масло, курица, морковь, лапша, лук, чеснок, креветки, кокосовое молоко, лемонграсс, какао, молоко, мёд и др.

**Заполненный план:** понедельник и вторник частично заполнены демо-рецептами.

## Хранение данных

Ключи localStorage:
- `mealflow_recipes` — массив рецептов
- `mealflow_ingredients` — база ингредиентов
- `mealflow_plan` — объект `{ "mon": { "breakfast": [...], "lunch": [...], ... }, ... }`
- `mealflow_shopping_checked` — объект `{ "ингредиент_единица": true }`

Автосохранение при каждом изменении.
