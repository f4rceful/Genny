# Характеристики (Features)

- Название приложения — «TaskFlow»
- Тёмная тема по умолчанию
- Три колонки на доске: «К выполнению», «В работе», «Выполнено»
- Карточка задачи показывает название, краткое описание и кнопки смены статуса (← →)
- Кнопка удаления задачи — крестик в правом верхнем углу карточки
- Подтверждение удаления через стандартный `confirm()` браузера
- Данные хранить в localStorage — задачи сохраняются между сессиями
- Фильтр по статусу — кнопки-таблетки над доской («Все», «К выполнению», «В работе», «Выполнено»)
- Все надписи на русском языке
- Счётчик задач в заголовке каждой колонки
